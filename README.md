Este é um pequeno projeto em C# feito para gerar senhas personalizadas.
O usuário pode escolher:

Tamanho da senha

Se inclui letras maiúsculas

Se inclui letras minúsculas

Se inclui números

Se inclui símbolos

O programa valida as entradas e só continua quando o usuário digita opções válidas.
Ao final, ele monta a lista de caracteres permitidos e gera uma senha aleatória com base nas escolhas feitas.

Projeto simples, mas útil para treinar lógica, estrutura de repetição, validação de entrada e manipulação de strings no C#.
